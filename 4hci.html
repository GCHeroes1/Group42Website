Final prototype

As a chatbot, the fundamental idea behind the chatbot remained the same- an IM interface that a GP can message into to ask questions. The difference in our prototypes is how they are accessed shown below; our first prototype stores the chatbot in a standalone application that must be downloaded. The second prototype is a web app which is accessed through a login screen.

Prototype 1 - Standalone application

Users are required to download and install the chatbot as an application on their device. They do this from the RCGP website.

Upon installation, they can open the application and login. If they donâ€™t have a login, there is an option to register which opens the register account web page in a browser on the RCGP website.

Prototype 2 - Web application

New users are required to register first at the page on the left where they put in their NHS email and create a password for their account.

If they have an account already, they sign in using the screen on the right. This then takes you to the chatbot screen on the next slide.
