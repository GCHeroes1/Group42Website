Plan for term 2

Integrate LUIS into our chatbot - In order to be able to return search results for mispelt words as well as return similar suggestions for a keyword search

Implement a wider variety of queries - Upon extending our current codebase with further testing we intend to be able to implement a wide variety of queries that could be requested by our stakeholders

Integrate images/videos in chatbot - Integration of mutlimedia could help our stakeholders gain a better understanding from the brief response

Visualise projects on a map - In order to pinpoint the location origin of the query results

Extend our database to include more data sources - We intend to include medical journals within the database to improve the querying abilities of our users

Allow for local storage of database - the ability to copy out database into a local copy would allow for any allowed institute to utilise our database for their own projects

Interface for adding/updating projects manually - this would be done in a way that a standard user would not be able to carry out the same modifications
