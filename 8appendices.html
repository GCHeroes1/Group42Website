Deployment manual

Pre-Requisites

  • Git installed on cmd or bash terminal
  • .NET Core Runtime & SDK (dotnet version 2.2 - https://aka.ms/dotnet-core-applaunch?framework=Microsoft.AspNetCore.App&framework_version=2.1.1&arch=x64&rid=ubuntu.18.04-x64)
  • Microsoft Bot Framework Emulator

Deployment Instructions

First, we need to start the server on the localhost for chatbot clients to connect to:

  1. Open your terminal/CMD and type: "git clone https://github.com/addybongo/entrepreneurChatbot [directoryNameToCloneTo]"
  2. Navigate to the directory you cloned into using cd ("cd path/to/directory")
  3. Run command "cd prototypeVer1" to enter the project directory
  4. Run command "dotnet run"

If this is successful, the result should be this message displayed on the command line


Second, we need to connect to our chatbot as a client. As we are still in development, our chatbot isn't connected to an IM interface like Facebook or Whatsapp. We instead are using the Microsoft Bot Emulator, to connect to our server. We do this using the following instructions:

  1. Open the Bot Emulator
  2. Click the Open Bot button on the front panel and it should show an 'Open Bot' prompt
  3. In the Bot URL box, type in: http://localhost:3978/api/messages
  4. Leave all other options and click 'Connect'

On a successful connection, the bot should begin running in the emulator and you can begin using it. The screen should look identical to the one below:
