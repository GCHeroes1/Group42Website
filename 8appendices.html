User manual

Our application is accessible via "https://ixnsearch.z22.web.core.windows.net/"

In order to start the chatbot, enter any message e.g. "hi", the chatbot will then prompt further for you to give another response. You can type "help" to view instructions on how to use the chatbot, same help is provided below:

Example commands:

Searching for projects:

Project keywords - e.g. "show me projects that use AI"

Location - e.g. "List all projects in London"

Organization name - e.g. "show me projects by IBM"

Contact - e.g. "List projects led by Joseph Connor"

Devices used - e.g. "Show projects that use IOS devices"

See more results:

To see the more results from your search, simply ask the chatbot!

e.g. "show me more projects" or "are there more results?"

Learn more about a project:

Once you’ve searched for a project, click on it to view more detailed information. You can click the buttons on the cards to view individual sections, or ask the chatbot for information you can't find e.g. "show me the ethics assessment"
